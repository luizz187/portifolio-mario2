# Portfólio do Mario 3D

Um site inspirado no universo do Super Mario em 3D, com foco em design moderno, acessibilidade e responsividade.

## Conteúdo
- `index.html` – Página principal
- `styles.css` – Estilo visual Mario 3D
- `script.js` – Acessibilidade + animações
- `img/` – Pasta para imagens (vazia)

## Instruções
1. Adicione suas imagens na pasta `img/`
2. Configure seu GitHub Pages em:
   **Settings → Pages → Deploy from main branch**
3. Pronto! Seu site estará online.

## Observação
As imagens devem ter estes nomes:
- `mario-inicio.png`
- `mario-bg.jpg`
- `odyssey.png`
- `3dworld.png`
- `bowser.png`
